# stm32-blinkLR1|
Лабораторна робота 1. Технологія програмування портів вводу-виводу загального
призначення на основі прототипу операційної системи нескінченного циклу опитування
Обладнання: Налагоджувальна плата STM32F103C8T6, ST-LINK V2 SWIM програматор STM8 STM32, CubeIDE
Мета: Ознайомитися з принципами програмування GPIO STM32F411 через регістрові 
структури, реалізувати нескінченний цикл опитування кнопки та керування світлодіодами
Варіанти завдання
1. Запрограмувати миготіння світлодіода PD12 з частотою 1 Гц (50% тривалість)
